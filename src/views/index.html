GDPRNU is a web application where companies and individuals can check their compliance in regards to the General Data Protection Regulation.

The GDPR(General Data Protection Regulation) is a upcoming European law that aims to set a baseline of standards for all the businesses that that process and handle personal data.

The platform serves as a 2 way communication channel, where visitors can find out more about the regulation in general, while in the same time they can interact with a real time questionnaire.


The questionarire determines the level of GDPR readiness based on the choices picked.


At the end of the questionnaire, the calculation engine determines a individual score, that gets presented to the visitors.


The application offers the possibility of providing a callback address where the administrators will get in touch with the clients.

Headless CMS integration - html5/css - javascript - calculation engine - automation
